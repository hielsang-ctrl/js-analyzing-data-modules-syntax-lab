# Documentation Index

This directory centralizes project documentation and points contributors to the main lab materials.

## Core Files

- `README.md`: lab scenario, setup, and implementation instructions
- `index.js`: current solution implementation
- `test/index.test.js`: Jest coverage for expected behavior
- `CONTRIBUTING.md`: workflow for version control, reviews, and maintenance

## Update Checklist

- Confirm the lab instructions still match the code and tests.
- Run `npm test` after code or documentation changes that affect expected behavior.
- Add new examples or support notes here when the lab grows beyond a single README.
